# Train Consist Management App

A Java application for managing train consists, including locomotives and bogies, with functionality to map bogie capacities and update train configurations.

## Features
- Create and manage train consists with locomotives and bogies
- Map bogies to specific capacities (passenger or cargo)
- Update or remove bogies dynamically
- Modular design for future enhancements

## Tech Stack
- Java
- IntelliJ IDEA
